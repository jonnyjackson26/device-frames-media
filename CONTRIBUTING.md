# Contributing

Thanks for helping expand the device frame dataset.

## Quick Start
1. Fork the repo and create a branch.
2. Add or update **raw** frame PNGs in `device-frames-raw/{device-type}/{device-model}/{variant}/`.
3. Open a pull request with a clear summary of what was added or changed.

Once merged, a github actions workflow will process the devices, update the readme list of devices. *DO NOT* run process_frames.py or update anything else.
